## ğŸ“ **PULL SEQUENCE**

### **Normal Mode:**

1. **Pull** - Start with 2 heads active (Frozen + Flaming recommended)
2. **Kill First Head** - Focus Frozen Head
3. **Rampage 1** - Stack, heal through 20 seconds
4. **Active Heads** - Tanks split, face away, begin swapping
5. **Kill Second Head** - Focus Flaming Head
6. **Rampage 2** - Stack, handle Torrent of Ice if targeted
7. **Ongoing** - Kill heads in rotation, never let one gain 3+ Hydra Frenzy
8. **Void Zone Management:**
   - Cinders: Clear ice, dispel in safe spots
   - Torrent: Kite away, clear fire void zones
   - Acid Rain: Move 15-20 yards
9. **Final Push** - Bloodlust during 6th/7th Rampage, burn last head

### **Heroic Mode:**

1. **Pull** - Same as Normal
2. **Never Kill Venomous** - Three Acid Rains too difficult
3. **Kill Order** - Rotate: Frozen â†’ Flaming â†’ Arcane (repeat)
4. **Cinders Management** - Dispel immediately, void zone every 3 seconds
5. **Ice Expansion** - Monitor expanding ice void zones, use Cinders to clear if needed
6. **Nether Wyrms** - Kill during Rampage, interrupt Nether Spike/Suppression
7. **Three Acid Rains** - If Venomous alive, spread more for three impacts
8. **DPS Priority** - Heads > Wyrms always
9. **Final Push** - Same as Normal, Bloodlust late
